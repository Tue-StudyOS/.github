# StudOS                                                                                                                                                                                            
                                                                                                                                                                                                       
   StudOS is a course project at the University of Tübingen led by Peter Gehler. This organization hosts the repositories, tools, and documentation developed as part of the project.                  
                                                                                                                                                                                                       
   ## About                                                                                                                                                                                            
                                                                                                                                                                                                       
   StudOS brings together software, experiments, and supporting materials created during the course.                                                                                                   
   The goal of this organization is to provide a central place for all related repositories.                                                                                                           
                                                                                                                                                                                                       
   ## Repositories                                                                                                                                                                                     
                                                                                                                                                                                                       
   Current repositories in this organization include:                                                                                                                                                  
                                                                                                                                                                                                       
   - **StudyPlanner** – planning and scheduling functionality for StudOS                                                                                                                               
   - More repositories will be added over time                                                                                                                                                         
                                                                                                                                                                                                       
   ## Purpose                                                                                                                                                                                          
                                                                                                                                                                                                       
   This organization is intended to:                                                                                                                                                                   
                                                                                                                                                                                                       
   - keep all StudOS repositories in one place                                                                                                                                                         
   - make project structure easier to understand                                                                                                                                                       
   - centralize documentation and collaboration                                                                                                                                                        
   - support future development across multiple repositories                                                                                                                                           
                                                                                                                                                                                                       
   ## Getting Started                                                                                                                                                                                  
                                                                                                                                                                                                       
   Each repository contains its own setup instructions, development workflow, and technical documentation.                                                                                             
                                                                                                                                                                                                       
   To get started, open the repository you are interested in and follow its README.
