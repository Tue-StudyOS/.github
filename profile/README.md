# StudyOS                                                                                                                                                                                            
                                                                                                                                                                                                       
   StudyOS is a course project at the University of Tübingen led by Peter Gehler. This organization hosts the repositories, tools, and documentation developed as part of the project.                  
                                                                                                                                                                                                       
   ## About                                                                                                                                                                                            
                                                                                                                                                                                                       
   StudyOS brings together software, experiments, and supporting materials created during the course.                                                                                                   
   The goal of this organization is to provide a central place for all related repositories.
